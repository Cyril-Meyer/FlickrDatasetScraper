# FlickrDatasetScraper

#### Usage
import module :  
```
import fds
```

search using text :  
```
fds.scrap(text='text', output_folder='text')
```

search using tags :  
```
fds.scrap(tags='tags', output_folder='tags')
```
:warning: There seems to be an issue with the API and sorting when using tag search. :warning:
